# MAWAR HALL RESERVATION WEBSITE

## Project Description

Mawar Hall Reservation Website is a simple web-based interface that allows users to view hall information and fill in a reservation form for event bookings such as weddings, meetings, and other functions.

This project focuses on the front-end design and user interface for a hall booking system. It demonstrates how a reservation website layout and booking flow can work without server-side processing.

## Features

* Home page with Mawar Hall introduction
* Hall information and event details
* Reservation form for users to enter booking information
* Simple navigation between pages
* Responsive design suitable for mobile devices

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

/mawar-hall-reservation

├── payment.html      -> Payment interface page
├── css/              -> Stylesheets
├── js/               -> JavaScript files
├── images/           -> Website images and assets
└── README.txt        -> Project documentation

## How to Run the Project

1. Download or clone the project from GitHub.
2. Extract the project folder if it is compressed.
3. Open the project folder.
4. Double click **index.html** or open it in any web browser (Chrome, Edge, Firefox).

No server or database is required to run this project.

## Purpose of the Project

This project was created as a practice project to demonstrate website interface design for a hall reservation system. It focuses on front-end development using HTML, CSS, and JavaScript.
